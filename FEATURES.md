# Features

## Major Features of the LENZ Project

- **Gmail OAuth Login**: Secure login using Gmail authentication.
- **OTP Verification**: One-Time Password verification for additional security.
- **Question Paper Upload**: Users can upload question papers easily.
- **AI Analysis with Topics Extraction**: Analyze question papers and extract relevant topics.
- **Key Notes Generation**: Generate key notes based on the analysis of the question papers.
- **Chat Assistant**: Interactive chat assistant for user queries.
- **Voice Input**: Support for voice commands and input.
- **Export to PDF/DOCX/JPEG**: Options to export documents in multiple formats.
- **Sharing**: Ability to share content with others easily.
- **Multi-Device Sync**: Synchronization across multiple devices.
- **Profiles**: User-specific profiles for personalized experience.
- **Glassmorphism UI**: Modern UI design using glassmorphism aesthetics.
